This was my first Axure Project. If you take a look at it, you can see that I did not yet understand page dimensions, and in addition, I was not aware of the useful interaction tools that axure offered the user. This was created for my UX Design class where the goal was to recreate the mobile app "Coffee Meets Bagel" in Axure.

This was the first team project that I worked on using Axure's team collaboration tools. This again was for my UX Design class, but this time we were tasked with redoing the "Coffee Meets Bagel" application, but with interactive elements. Our Group was able to create this, which is able to cycle through all of the pages that the application has, from the like to the dislike buttons.

This was again for my UX Design class, and this is the first time that we had to create a persona for use in developing our target audience further. We created this persona based off of possible characteristics one would find in a regular NJIT student.

This little project was actually mostly done for fun. The original idea behind this project was that for my Technical Writing class we were supposed to write a project proposal.
